Podemos procurar pelo nome ou locator. No locator usamos a URL(uniform resource locator) Ou seja, é um tipo de localizador pelo endereço Ela precisa de componentes: Obrigatórios: - Protocolo (https://) - Domínio (.com.br) * os dois são parte de uma URL Opcionais: - Subdomíno (www) - Path (/blog - o caminho) - Parametros (?v=vpYct2npKD8) - Porta (:3333) - Âncora (#algumlugar - dentro do documento) * Opcionais não significa se vai encontrar ou não * significa que ele vai fazer díferença para encontrar ou não obs: o domíno pode ser tanto pelo nome, quanto pelo IP http sem definir uma porta, por trás ele usa a porta: :80 https: porta :443
