um proxy é um servidor que age como um intermediário para requisições de clientes solicitando recursos de outros servidores. * Fica entre o cliente e o servidor * Ajudam a fazer o transorte de dados Exemplo: Roteador Wi-fi Funções: cache (devolver informações mais rápido) Filtro (antivirus ou controle parental) Load Balancing (distribuição de carga) Autenticação Autorização
